À propos de notre projet

Cette application web fait partie du projet ft_transcendence du cursus 42. Elle a été créée pour développer une application full-stack avec des fonctionnalités en temps réel, en se concentrant à la fois sur l'apprentissage et l'application pratique des compétences de développement web.

Objectifs


L'objectif principal de ce projet est de construire une application web complète et interactive qui intègre :
- Interface utilisateur frontend pour des interactions fluides et réactives
- Services backend avec des APIs sécurisées
- Gestion de base de données pour un stockage fiable des données
- Fonctionnalités en temps réel pour des mises à jour et une communication en direct
- Authentification et gestion des utilisateurs

Technologies utilisées


Frontend : React
- Pour construire des interfaces utilisateur dynamiques et réactives

Backend : Express / Nest / Django
- Pour gérer les requêtes API et la logique métier

Base de données : PostgreSQL / SQLite / MongoDB
- Pour le stockage persistant des données

Docker
- Pour simplifier le déploiement et garantir des environnements cohérents

Équipe et flux de travail


Ce projet a été développé de manière collaborative, en suivant les meilleures pratiques en matière de contrôle de version, de qualité du code et de gestion de projet. Notre équipe a mis l'accent sur le code propre, la conception modulaire et la communication efficace pour réaliser une application fonctionnelle et maintenable.
