Politique de confidentialité

Avis de confidentialité - Pong Evolution


Dernière mise à jour : 15 février 2026

Pong Evolution est un projet académique développé dans le cadre du cursus 42. L'application fonctionne localement et n'est pas accessible publiquement en ligne.

Données collectées


L'application peut traiter les données suivantes :
- Adresse e-mail
- Nom d'utilisateur
- Mot de passe crypté
- Statistiques de jeu
- Messages de chat
- Données techniques (telles que l'adresse IP dans l'environnement local)

Finalité


Les données sont traitées exclusivement pour :
- Permettre la création de comptes
- Activer le jeu multijoueur
- Fournir une fonctionnalité de chat
- Stocker les statistiques de jeu

Partage de données


Aucune donnée personnelle n'est partagée avec des tiers. Le projet n'utilise pas de cookies, de publicité ou de services de suivi externes.

Conservation des données


Les données sont stockées localement dans l'environnement du projet et peuvent être supprimées à tout moment lorsque l'environnement du projet est réinitialisé ou supprimé.

Nature éducative


Ce projet est développé uniquement à des fins éducatives et n'est pas destiné à un usage commercial.
