Sobre el nostre projecte

Aquesta aplicació web forma part del projecte ft_transcendence del currículum de 42. Es va crear per desenvolupar una aplicació full-stack amb funcionalitats en temps real, centrant-se tant en l'aprenentatge com en l'aplicació pràctica d'habilitats de desenvolupament web.

Objectius


L'objectiu principal d'aquest projecte és construir una aplicació web completa i interactiva que integri:
- Interfície d'usuari frontend per a interaccions fluides i responsives
- Serveis backend amb APIs segures
- Gestió de base de dades per a emmagatzematge fiable de dades
- Funcionalitats en temps real per a actualitzacions i comunicació en viu
- Autenticació i gestió d'usuaris

Tecnologies utilitzades


Frontend: React
- Per construir interfícies d'usuari dinàmiques i responsives

Backend: Express / Nest / Django
- Per gestionar sol·licituds d'API i lògica de negoci

Base de dades: PostgreSQL / SQLite / MongoDB
- Per a emmagatzematge persistent de dades

Docker
- Per simplificar el desplegament i garantir entorns consistents

Equip i flux de treball


Aquest projecte va ser desenvolupat de forma col·laborativa, seguint les millors pràctiques en control de versions, qualitat de codi i gestió de projectes. El nostre equip va emfatitzar el codi net, el disseny modular i la comunicació efectiva per aconseguir una aplicació funcional i mantenible.
