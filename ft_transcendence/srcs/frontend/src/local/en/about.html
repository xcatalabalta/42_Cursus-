About our project

This web application is part of the ft_transcendence project from the 42 curriculum. It was created to develop a full-stack application with real-time functionalities, focusing on both learning and practical application of web development skills.

Objectives


The main goal of this project is to build a complete and interactive web application that integrates:
- Frontend user interface for smooth and responsive interactions
- Backend services with secure APIs
- Database management for reliable data storage
- Real-time features for live updates and communication
- User authentication and management

Technologies Used


Frontend: React
- For building dynamic and responsive user interfaces

Backend: Express / Nest / Django
- To handle API requests and business logic

Database: PostgreSQL / SQLite / MongoDB
- For persistent data storage

Docker
- To simplify deployment and ensure consistent environments

Team & Workflow


This project was developed collaboratively, following best practices in version control, code quality, and project management. Our team emphasized clean code, modular design, and effective communication to achieve a functional and maintainable application.
