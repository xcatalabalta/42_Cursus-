Privacy Notice - Pong Evolution


Last updated: February 15, 2026

Pong Evolution is an academic project developed within the 42 curriculum. The application runs locally and is not publicly accessible online.

Data Collected


The application may process the following data:
- Email address
- Username
- Encrypted password
- Game statistics
- Chat messages
- Technical data (such as IP address within the local environment)

Purpose


Data is processed exclusively to:
- Allow account creation
- Enable multiplayer gameplay
- Provide chat functionality
- Store game statistics

Data Sharing


No personal data is shared with third parties. The project does not use cookies, advertising, or external tracking services.

Data Retention


Data is stored locally within the project environment and may be deleted at any time when the project environment is reset or removed.

Educational Nature


This project is developed for educational purposes only and is not intended for commercial use.
