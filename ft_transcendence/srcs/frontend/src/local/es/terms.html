Términos de servicio

Condiciones de uso - Pong Evolution


Última actualización: 15 de febrero de 2026

Naturaleza del proyecto


Pong Evolution es un proyecto académico desarrollado dentro del currículo de 42 (Transcendence). El Servicio no está alojado públicamente en internet y opera únicamente dentro de un entorno local (por ejemplo, mediante Docker). Se proporciona exclusivamente con fines educativos y no es un producto comercial.

Aceptación de los términos


Al acceder o utilizar el Servicio, Usted acepta estar sujeto a estas Condiciones de uso. Si no está de acuerdo con estos Términos, no debe utilizar el Servicio.

Cuentas de usuario


El Servicio permite a los usuarios crear cuentas con fines de juego. Usted acepta:
- Proporcionar información precisa al crear una cuenta
- Mantener sus credenciales de acceso confidenciales
- No suplantar a otros usuarios
- Naturaleza temporal de las cuentas
- Las cuentas de usuario y los datos relacionados se almacenan dentro del entorno local del proyecto.

Cuando el entorno Docker se detiene, reinicia o elimina:
- Las cuentas pueden ser eliminadas permanentemente
- El historial de juegos y los datos de chat pueden borrarse
- No se garantiza la persistencia de los datos
- Al utilizar el Servicio, Usted reconoce y acepta que todas las cuentas y datos pueden ser temporales.

Uso aceptable


Usted acepta no:
- Utilizar el Servicio con fines ilícitos
- Intentar explotar errores o vulnerabilidades
- Interferir con el funcionamiento normal del juego
- Cargar o enviar código malicioso a través del chat
- Acosar o abusar de otros usuarios
- El Servicio está destinado a un uso respetuoso y educativo.

Propiedad intelectual


Este proyecto está desarrollado con fines educativos. Salvo que se indique lo contrario, la lógica del juego, el diseño y el código forman parte del proyecto académico y no pueden reutilizarse ni redistribuirse sin el permiso del/de los autor/es del proyecto.

Sin uso comercial


El Servicio:
- No cobra tarifas
- No vende bienes virtuales
- No proporciona servicios de pago
- No está destinado a la explotación comercial

Renuncia de garantía


El Servicio se proporciona "TAL CUAL" con fines educativos. No garantizamos que:
- El Servicio esté libre de errores
- El Servicio funcione siempre correctamente
- Los datos se almacenen permanentemente
- El juego sea ininterrumpido
- El Servicio puede contener errores o problemas técnicos.

Limitación de responsabilidad


En la máxima medida permitida por la ley aplicable, el autor del proyecto no será responsable de:
- Pérdida de datos
- Pérdida de cuentas
- Mal funcionamiento técnico
- Daños resultantes del uso o la imposibilidad de usar el Servicio
- Dado que se trata de un proyecto académico no comercial, no se asume ninguna responsabilidad financiera.

Terminación


El acceso al Servicio puede ser suspendido o terminado en cualquier momento por: Violación de estos Términos Uso indebido de la plataforma Comportamiento disruptivo Además, el Servicio puede ser descontinuado en cualquier momento como parte del ciclo de vida académico.

Ley aplicable


Estos Términos se rigen por las leyes de España.

Cambios en estos términos


Estos Términos pueden actualizarse si el proyecto evoluciona. La última versión será siempre la disponible dentro del entorno del proyecto.
