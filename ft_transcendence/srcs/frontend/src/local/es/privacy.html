Política de privacidad

Aviso de privacidad - Pong Evolution


Última actualización: 15 de febrero de 2026

Pong Evolution es un proyecto académico desarrollado dentro del currículo de 42. La aplicación se ejecuta localmente y no es accesible públicamente en línea.

Datos recopilados


La aplicación puede procesar los siguientes datos:
- Dirección de correo electrónico
- Nombre de usuario
- Contraseña cifrada
- Estadísticas del juego
- Mensajes de chat
- Datos técnicos (como la dirección IP dentro del entorno local)

Finalidad


Los datos se procesan exclusivamente para:
- Permitir la creación de cuentas
- Habilitar el juego multijugador
- Proporcionar funcionalidad de chat
- Almacenar estadísticas del juego

Compartir datos


No se comparten datos personales con terceros. El proyecto no utiliza cookies, publicidad ni servicios de seguimiento externos.

Retención de datos


Los datos se almacenan localmente dentro del entorno del proyecto y pueden eliminarse en cualquier momento cuando el entorno del proyecto se reinicie o elimine.

Naturaleza educativa


Este proyecto se desarrolla únicamente con fines educativos y no está destinado a uso comercial.
