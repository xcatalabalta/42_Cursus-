Sobre nuestro proyecto

Esta aplicación web forma parte del proyecto ft_transcendence del currículo de 42. Se creó para desarrollar una aplicación full-stack con funcionalidades en tiempo real, centrándose tanto en el aprendizaje como en la aplicación práctica de habilidades de desarrollo web.

Objetivos


El objetivo principal de este proyecto es construir una aplicación web completa e interactiva que integre:
- Interfaz de usuario frontend para interacciones fluidas y responsivas
- Servicios backend con APIs seguras
- Gestión de base de datos para almacenamiento confiable de datos
- Funcionalidades en tiempo real para actualizaciones y comunicación en vivo
- Autenticación y gestión de usuarios

Tecnologías utilizadas


Frontend: React
- Para construir interfaces de usuario dinámicas y responsivas

Backend: Express / Nest / Django
- Para manejar solicitudes de API y lógica de negocio

Base de datos: PostgreSQL / SQLite / MongoDB
- Para almacenamiento persistente de datos

Docker
- Para simplificar el despliegue y garantizar entornos consistentes

Equipo y flujo de trabajo


Este proyecto fue desarrollado de forma colaborativa, siguiendo las mejores prácticas en control de versiones, calidad de código y gestión de proyectos. Nuestro equipo enfatizó el código limpio, el diseño modular y la comunicación efectiva para lograr una aplicación funcional y mantenible.
